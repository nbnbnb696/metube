# MeTube - Video Sharing Platform

A YouTube-like video sharing web application built with HTML, CSS, and JavaScript.

## Features

### Core Features
- ✅ Video grid display with thumbnails
- ✅ Video player page with controls
- ✅ Video upload interface
- ✅ Search functionality
- ✅ Responsive design
- ✅ Like/Dislike system
- ✅ Comments section
- ✅ Subscribe button
- ✅ Video recommendations
- ✅ Navigation sidebar

### Pages
1. **Home Page (index.html)** - Browse all videos in a grid layout
2. **Video Player (video.html)** - Watch videos with full details
3. **Upload Page (upload.html)** - Upload new videos

## File Structure
```
metube/
├── index.html          # Home page
├── video.html          # Video player page
├── upload.html         # Upload page
├── styles.css          # All styles
├── script.js           # Home page logic
├── video.js            # Video player logic
├── upload.js           # Upload page logic
└── README.md           # Documentation
```

## How to Run

1. Open `index.html` in your web browser
2. Browse videos on the home page
3. Click any video to watch it
4. Use the Upload button to add new videos
5. Search for videos using the search bar
